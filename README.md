# E-Commerce RESTFUL API
Backend for an ecommerce website/app
This is a mock RESTFUL API built with Node, Express and MySQL. It is Pseudo-real data for e-commerce website. 


# Endpoints

Get all products
```
/api/v1/product
```

Get a single product
```
/api/v1/product/:id
```

Post a product
```
/api/v1/product
```
